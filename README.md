# Ocean Defi

> Ocean-themed decentralized exchange

## Features

- AMM swaps
- Liquidity mining
- NFT staking
- Governance

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/bomniop/ocean-defi.git
cd ocean-defi
npm install
npm run dev
```
